Cadre

De multe ori se dereaste ca, in aceeasi instanta a browserului sa poata fi afisate mai multe pagini web in acelasi timp. In acest scop este deosebit de utila divizarea paginii web in mai multe zone numite cadre (frames). Pentru divizarea paginii web se foloseste eticheta de inceput de definire a cadrelor, FRAMESET, se definesc cadrele cu etichete FRAME iar setarile cadrelor se incheie cu eticheta /FRAMESET.
  • Eticheta FRAMESET si atributele ei
    • Atributul cols
    • Atributul rows
    • Imbricarea etichetelor FRAMESET
  • Eticheta FRAME si atributele ei
    • Atributul name
    • Atributul src
    • Atributul frameborder
    • Atributul bordercolor
    • Atributul scrolling
    • Atributul noresize
