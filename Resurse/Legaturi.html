Legaturi

Legaturile, numite si link-uri sau hyperlink-uri, sunt zonele active din paginile web cuprinse intre etichetele A (Anchor) si respectiv /A, zone sensibile la click cu butonul stang al mouse-ului. Un click de mouse pe o zona activa este interpretat ca o cerere pentru o pagina web aflata pe un anumit server din Internet. Zonele active pot fi text (care astfel este transformat in hipertext) sau o imagine sau alt obiect (fapt care transforma obiectul in hipermedia). Asadar, legaturile sunt mijloacele prin care utilizatorul poate naviga prin Internet printre paginile web ale siturilor.
